About the Project

A simple hub for comparing mods and learning what gives the best value.

What this site does

  • Helps compare mod cost vs estimated gains (Performance Per Dollar).
  • Organizes common bolt-ons and dyno examples for quick reference.
  • Supports a clean, beginner-friendly layout for mobile and desktop.

How to use it

Start with the calculator to estimate value, then check the Gallery to see examples of parts, dynos, and common upgrade paths.
